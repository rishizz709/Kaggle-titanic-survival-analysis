# Titanic Survival Analysis

## Overview
Exploratory data analysis and a simple logistic regression model predicting passenger survival on the Titanic.

## Dataset
[Titanic - Machine Learning from Disaster (Kaggle)](https://www.kaggle.com/competitions/titanic)

## Notebook
[View on Kaggle]([your-kaggle-notebook-link-here](https://www.kaggle.com/code/rishibusineni709/notebook45096f100b))

## Results
Logistic regression model achieved ~80.4% accuracy on the test split.

## What I Learned
- How to clean missing data (Age, Cabin, Embarked)
- Basic EDA and visualization with seaborn
- Building and evaluating a simple classification model
